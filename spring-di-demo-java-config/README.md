# Spring Dependency Injection Demo - Java Config

## Objective
To create a simple Spring application that demonstrates Dependency Injection (DI) using Java-based configuration instead of XML.

## Project Structure
This module demonstrates:
- Java-based configuration using `@Configuration` and `@Bean` annotations
- Dependency injection without XML configuration
- Bean lifecycle management
- Different types of injection (constructor, setter)

## Technologies
- Spring Framework
- Java
- Maven

## Setup Instructions
1. Configure Maven dependencies
2. Create configuration classes with `@Configuration`
3. Define beans using `@Bean` annotations
4. Run the application
