# Hibernate Student CRUD Application

## Objective
To develop a Hibernate application that performs CRUD operations on a Student entity stored in a MySQL database using Hibernate ORM.

## Project Structure
This module demonstrates:
- Hibernate ORM configuration
- Entity mapping using annotations
- CRUD operations (Create, Read, Update, Delete)
- Session management
- Database connectivity with MySQL

## Technologies
- Hibernate ORM
- MySQL Database
- Java
- Maven

## Setup Instructions
1. Configure MySQL database connection in `hibernate.cfg.xml`
2. Create Student entity class with proper annotations
3. Implement DAO (Data Access Object) for CRUD operations
4. Test all CRUD operations
